ChatGPT API by Ashlynn Repository

Explore all GPT models and use them with our easy-to-use API.

Available API Endpoints

GET Request

Query a question with a Default GPT-4

GET https://chatgpt.ashlynn.workers.dev/?question=Hello

GET Request

GPT-4.0 with real timedata.

GET https://chatgpt.ashlynn.workers.dev/gptweb/?question=today%27s%20date%20and%20time%20in%20new%20delhi%20India

GET /gpt-3.5-turbo

Query using the GPT-3.5 turbo model.

GET https://chatgpt.ashlynn.workers.dev/?question=What+is+AI?&model=gpt-3.5-turbo
